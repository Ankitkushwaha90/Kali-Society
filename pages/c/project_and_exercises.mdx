## Basic Programs for Practice

Start your journey with these simple yet effective basic programs to get hands-on experience with fundamental concepts of C programming. These programs cover topics such as arithmetic operations, conditional statements, loops, arrays, and functions.

```c
#include <stdio.h>

// Program to find the factorial of a number
int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    } else {
        return n * factorial(n - 1);
    }
}

int main() {
    int num = 5;
    printf("Factorial of %d is %d\n", num, factorial(num));
    return 0;
}
```
### Intermediate Level Projects
Once you're comfortable with the basics, challenge yourself with these intermediate level projects that require a deeper understanding of C programming concepts. Build applications such as a simple calculator, a tic-tac-toe game, or a student management system to reinforce your knowledge and skills.

```c
#include <stdio.h>

// Program to implement a simple calculator
int main() {
    char operator;
    double num1, num2;

    printf("Enter an operator (+, -, *, /): ");
    scanf("%c", &operator);

    printf("Enter two operands: ");
    scanf("%lf %lf", &num1, &num2);

    switch (operator) {
        case '+':
            printf("%.1lf + %.1lf = %.1lf\n", num1, num2, num1 + num2);
            break;
        case '-':
            printf("%.1lf - %.1lf = %.1lf\n", num1, num2, num1 - num2);
            break;
        case '*':
            printf("%.1lf * %.1lf = %.1lf\n", num1, num2, num1 * num2);
            break;
        case '/':
            if (num2 != 0)
                printf("%.1lf / %.1lf = %.1lf\n", num1, num2, num1 / num2);
            else
                printf("Error! Division by zero.\n");
            break;
        default:
            printf("Error! Invalid operator.\n");
    }

    return 0;
}
```

### Exercises with Solutions
Test your skills with these exercises covering a wide range of C programming concepts. Practice solving problems related to arrays, strings, pointers, and data structures. Each exercise comes with a solution to help you verify your answers and learn from the implementation.

```c

#include <stdio.h>

// Exercise: Find the maximum element in an array
int findMax(int arr[], int size) {
    int max = arr[0];
    for (int i = 1; i < size; i++) {
        if (arr[i] > max) {
            max = arr[i];
        }
    }
    return max;
}

int main() {
    int numbers[] = {23, 45, 12, 67, 89, 34};
    int size = sizeof(numbers) / sizeof(numbers[0]);
    int max = findMax(numbers, size);
    printf("Maximum element in the array: %d\n", max);
    return 0;
}
```
