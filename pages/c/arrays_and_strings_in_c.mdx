# Arrays and Strings in C Programming

Arrays and strings are essential data structures in C programming that allow you to store and manipulate collections of data efficiently. In this tutorial, we'll cover the basics of arrays and strings in C.

## Introduction to Arrays

An array is a collection of elements of the same data type stored in contiguous memory locations. Arrays provide a convenient way to store and access multiple values of the same type. Here's how you declare and initialize an array in C:

```c
// Declare and initialize an array of integers
int numbers[5] = {1, 2, 3, 4, 5};
```
### One-dimensional and Multi-dimensional Arrays
- One-dimensional Arrays: Arrays with a single row or column. They are commonly used to store lists of values.
- Multi-dimensional Arrays: Arrays with multiple rows and columns. They are often used to represent matrices and tables.
```c
// Declare and initialize a two-dimensional array
int matrix[3][3] = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
```
### Array Initialization
You can initialize an array either at the time of declaration or later in the program using loops or assignment statements. Here's an example of array initialization using a loop:

```c
int numbers[5];
for (int i = 0; i < 5; i++) {
    numbers[i] = i + 1;
}
```
### Array Manipulation
Arrays in C can be manipulated using various operations such as accessing elements, modifying elements, and performing computations. Here's an example of accessing and modifying elements in an array:

```c
int numbers[5] = {10, 20, 30, 40, 50};
printf("Element at index 2: %d\n", numbers[2]); // Output: 30
numbers[3] = 60; // Modify element at index 3
```
### Introduction to Strings
A string in C is a sequence of characters stored in contiguous memory locations, terminated by a null character (`'\0'`). Strings in C are represented as arrays of characters.

```c
char greeting[] = "Hello, World!";
```
### String Manipulation Functions
C provides several built-in functions for manipulating strings, such as `strlen()`, `strcpy()`, `strcat()`, `strcmp()`, and more. These functions allow you to perform common string operations efficiently.

```c
#include <stdio.h>
#include <string.h>

int main() {
    char str1[] = "Hello";
    char str2[] = "World";
    strcat(str1, str2); // Concatenate str2 to str1
    printf("Concatenated string: %s\n", str1); // Output: HelloWorld
    return 0;
}
```
