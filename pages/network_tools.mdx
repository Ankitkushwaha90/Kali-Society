# Introduction
Welcome to the Networking Tools for Scanning tutorial! Network scanning is a crucial aspect of network administration, security assessment, and troubleshooting. In this tutorial, we'll explore various networking tools used for scanning networks, discovering hosts, identifying open ports, and assessing network security.

## 1. Nmap (Network Mapper)
Nmap is a powerful open-source tool used for network exploration and security auditing. It can discover hosts, services, and vulnerabilities on computer networks. Here are some key features and commands:

- `nmap <target>`: Perform a basic scan on the specified target.
- `nmap -sP <target>`: Ping scan to determine which hosts are online.
- `nmap -p <port(s)> <target>`: Scan specific port(s) on the target.
- `nmap -A <target>`: Aggressive scan, includes OS detection and version detection.

## 2. Netcat (nc)
Netcat is a versatile networking utility that can read and write data across network connections using TCP or UDP protocols. It's often used for port scanning, file transfer, and creating network connections. Here are some key features and commands:

- `nc <host> <port>`: Connect to a specified host and port.
- `nc -l -p <port>`: Listen for incoming connections on a specified port.
- `nc -zv <host> <start-port>-<end-port>`: Scan a range of ports for connectivity.

## 3. Masscan
Masscan is a high-speed TCP port scanner designed for large-scale network scans. It's capable of scanning the entire IPv4 address space in under 6 minutes. Here are some key features and commands:

- `masscan -p <port(s)> <target>`: Scan specific port(s) on the target.
- `masscan --top-ports <number> <target>`: Scan the top N most common ports.

## 4. Zenmap
Zenmap is the official graphical user interface (GUI) for Nmap. It provides an intuitive interface for visualizing and analyzing Nmap scan results. Here are some key features:

- Scan profiles: Pre-configured scan options for different scenarios.
- Host topology maps: Visual representation of network topology.
- Scripting engine integration: Execute Nmap scripts directly from the interface.

## 5. Wireshark
Wireshark is a powerful network protocol analyzer used for network troubleshooting, analysis, and development. It allows you to capture and interactively browse the traffic running on a computer network. Here are some key features:

- Live capture: Capture packets from live network interfaces.
- Packet filtering: Filter and search for specific packets based on various criteria.
- Protocol decryption: Decrypt encrypted protocols for analysis.
