#### To install MongoDB Community Edition using the apt package manager on Debian-based systems, you can follow these steps:

### Install Required Dependencies: If not already installed, install gnupg and curl:
```bash
sudo apt-get install gnupg curl
```
### Import MongoDB GPG Key: Run the following command to import the MongoDB public GPG key:
```bash
curl -fsSL https://www.mongodb.org/static/pgp/server-8.0.asc | \
sudo gpg -o /usr/share/keyrings/mongodb-server-8.0.gpg \
```
--dearmor
### Create Repository Configuration: Create a file named /etc/apt/sources.list.d/mongodb-org-8.0.list with the MongoDB repository information:
```bash
echo "deb [ signed-by=/usr/share/keyrings/mongodb-server-8.0.gpg ] http://repo.mongodb.org/apt/debian bookworm/mongodb-org/8.0 main" | sudo tee /etc/apt/sources.list.d/mongodb-org-8.0.list
```
Reload Package Database: Update the local package database:
```bash
sudo apt-get update
```
### Install MongoDB Packages: Install MongoDB Community Edition:
```bash
sudo apt-get install -y mongodb-org
```
### Optional: Pin Package Versions: To prevent unintended upgrades, you can pin the package at the currently installed version:
```bash
echo "mongodb-org hold" | sudo dpkg --set-selections
echo "mongodb-org-database hold" | sudo dpkg --set-selections
echo "mongodb-org-server hold" | sudo dpkg --set-selections
echo "mongodb-mongosh hold" | sudo dpkg --set-selections
echo "mongodb-org-mongos hold" | sudo dpkg --set-selections
echo "mongodb-org-tools hold" | sudo dpkg --set-selections
```

### Once installed, you can start, stop, and manage MongoDB using the systemctl commands. For example:

-  Start MongoDB: sudo systemctl start mongod
- Stop MongoDB: sudo systemctl stop mongod
- Restart MongoDB: sudo systemctl restart mongod
- Check MongoDB status: sudo systemctl status mongod
- Enable MongoDB to start on boot: sudo systemctl enable mongod


Remember to configure MongoDB according to your requirements, especially regarding security and network bindings, as mentioned in the MongoDB documentation.
