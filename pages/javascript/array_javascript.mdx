# JavaScript Arrays and Objects

JavaScript arrays and objects are essential data structures that allow you to organize and manipulate data efficiently. In this section, we'll explore how to create and manipulate arrays, work with array methods like map, filter, reduce, and understand objects and object-oriented programming concepts in JavaScript.

## Creating and Manipulating Arrays

### Creating Arrays

You can create an array in JavaScript using square brackets `[ ]` and adding elements separated by commas.

```javascript
var fruits = ["apple", "banana", "orange"];
```
### Manipulating Arrays
JavaScript provides various methods to manipulate arrays, such as adding or removing elements, accessing elements, and iterating over arrays.

```javascript
fruits.push("grape"); // Add an element to the end
fruits.pop(); // Remove the last element
console.log(fruits[0]); // Access the first element
```
### Working with Array Methods
JavaScript arrays come with built-in methods that make common tasks like mapping, filtering, and reducing data more manageable.

## map()
The `map()` method creates a new array by applying a function to each element in the original array.

```javascript
var numbers = [1, 2, 3, 4, 5];
var doubled = numbers.map(function(num) {
  return num * 2;
});
console.log(doubled); // Output: [2, 4, 6, 8, 10]
```
# filter()
The `filter()` method creates a new array with all elements that pass the test implemented by the provided function.

```javascript
var evenNumbers = numbers.filter(function(num) {
  return num % 2 === 0;
});
console.log(evenNumbers); // Output: [2, 4]
```
### reduce()
```
The `reduce()` method applies a function against an accumulator and each element in the array to reduce it to a single value.

```javascript
var sum = numbers.reduce(function(accumulator, currentValue) {
  return accumulator + currentValue;
}, 0);
console.log(sum); // Output: 15
```
## Understanding Objects and Object-Oriented Programming Concepts
### Objects
In JavaScript, objects are collections of key-value pairs and can contain properties and methods.

```javascript
var person = {
  name: "John",
  age: 30,
  greet: function() {
    console.log("Hello, my name is " + this.name);
  }
};

console.log(person.name); // Access property
person.greet(); // Call method
```
