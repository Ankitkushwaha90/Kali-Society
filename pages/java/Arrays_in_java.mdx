# Arrays in Java

Arrays are used to store multiple values of the same data type in a single variable. They provide a convenient way to work with collections of data in Java. In this tutorial, we'll cover the basics of arrays, including declaring and initializing arrays, accessing array elements, multidimensional arrays, and array manipulation.

## Declaring and Initializing Arrays

In Java, arrays are declared using square brackets `[]` after the data type. Here's how you can declare and initialize an array:

```java
// Declare and initialize an array of integers
int[] numbers = {1, 2, 3, 4, 5};

// Declare and initialize an array of strings
String[] names = {"Alice", "Bob", "Charlie"};
```
You can also declare an array without initializing it and then assign values later:

```java
// Declare an array of doubles
double[] prices;

// Initialize the array
prices = new double[5];
```
### Accessing Array Elements
Array elements are accessed using their index, which starts from 0. You can access individual elements of an array using square brackets `[]` with the index inside them:

```java
int[] numbers = {10, 20, 30, 40, 50};
System.out.println(numbers[0]); // Output: 10
System.out.println(numbers[2]); // Output: 30
```
### Multidimensional Arrays
Java supports multidimensional arrays, allowing you to create arrays of arrays. The most common type is the 2D array. Here's how you can declare and initialize a 2D array:

```java
// Declare and initialize a 2D array
int[][] matrix = {{1, 2, 3}, {4, 5, 6}, {7, 8, 9}};
```
You can access elements of a 2D array using two indices:

```java
System.out.println(matrix[0][1]); // Output: 2
System.out.println(matrix[2][2]); // Output: 9
```
### Array Manipulation
Java provides several methods for manipulating arrays, such as sorting, searching, and copying. Some commonly used methods include:

- Sorting: Use the `Arrays.sort()` method to sort elements of an array.
- Searching: Use methods like `Arrays.binarySearch()` to search for an element in an array.
- Copying: Use methods like `System.arraycopy()` or `Arrays.copyOf()` to copy elements from one array to another.
Example:

```java
int[] numbers = {5, 2, 8, 1, 9};

// Sorting the array
Arrays.sort(numbers);

// Searching for an element
int index = Arrays.binarySearch(numbers, 8);
System.out.println("Index of 8: " + index); // Output: Index of 8: 3

// Copying elements to a new array
int[] copiedArray = Arrays.copyOf(numbers, numbers.length);
```

If you're asking for an example of defining an array in Java within an object-oriented context, you can encapsulate the array within a class and define methods to interact with it. Here's a simple example:

```java
public class MyArray {
    private int[] array; // The array encapsulated within the class

    // Constructor to initialize the array
    public MyArray(int size) {
        array = new int[size];
    }

    // Method to get the length of the array
    public int length() {
        return array.length;
    }

    // Method to set a value at a specific index
    public void set(int index, int value) {
        if (index >= 0 && index < array.length) {
            array[index] = value;
        } else {
            System.out.println("Index out of bounds");
        }
    }

    // Method to get a value at a specific index
    public int get(int index) {
        if (index >= 0 && index < array.length) {
            return array[index];
        } else {
            System.out.println("Index out of bounds");
            return -1; // Or you can throw an exception
        }
    }

    // Method to print the array
    public void print() {
        for (int i = 0; i < array.length; i++) {
            System.out.print(array[i] + " ");
        }
        System.out.println();
    }

    // Other methods for array manipulation can be added here
}
```
Then, you can use this class in your main program like this:

```java
public class Main {
    public static void main(String[] args) {
        MyArray myArray = new MyArray(5); // Creating an instance of MyArray with size 5
        myArray.set(0, 10); // Setting value 10 at index 0
        myArray.set(1, 20); // Setting value 20 at index 1
        myArray.set(2, 30); // Setting value 30 at index 2
        myArray.print(); // Printing the array
        System.out.println("Value at index 1: " + myArray.get(1)); // Getting value at index 1
    }
}
```
This approach encapsulates the array within a class and provides methods to interact with it, ensuring better encapsulation and reusability. You can further extend this class to include more functionalities or customize it according to your requirements.

### ArrayList:
ArrayList is part of the Java Collections Framework and provides dynamic arrays in Java. It's similar to arrays but can dynamically grow and shrink.

```java
import java.util.ArrayList;

public class ArrayListExample {
    public static void main(String[] args) {
        // Creating an ArrayList
        ArrayList<String> arrayList = new ArrayList<>();

        // Adding elements
        arrayList.add("Apple");
        arrayList.add("Banana");
        arrayList.add("Orange");

        // Accessing elements
        System.out.println("Elements in ArrayList: " + arrayList);

        // Iterating through ArrayList
        for (String fruit : arrayList) {
            System.out.println(fruit);
        }
    }
}
```
### List Interface:
List is an interface in the Java Collections Framework that extends Collection and represents an ordered collection of elements.

```java
import java.util.List;
import java.util.ArrayList;

public class ListInterfaceExample {
    public static void main(String[] args) {
        // Creating a List using ArrayList
        List<String> list = new ArrayList<>();

        // Adding elements
        list.add("Red");
        list.add("Green");
        list.add("Blue");

        // Accessing elements
        System.out.println("Elements in List: " + list);

        // Iterating through List
        for (String color : list) {
            System.out.println(color);
        }
    }
}
```
### Vector:
Vector is a legacy class in Java that is synchronized and similar to ArrayList but slower. It is thread-safe, meaning multiple threads can safely access it concurrently.

```java
import java.util.Vector;

public class VectorExample {
    public static void main(String[] args) {
        // Creating a Vector
        Vector<String> vector = new Vector<>();

        // Adding elements
        vector.add("Cat");
        vector.add("Dog");
        vector.add("Bird");

        // Accessing elements
        System.out.println("Elements in Vector: " + vector);

        // Iterating through Vector
        for (String animal : vector) {
            System.out.println(animal);
        }
    }
}
```
