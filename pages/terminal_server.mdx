First, make sure you have Flask installed. You can install it using pip:

```bash
pip install flask
```
Next, create a directory for your project and navigate into it. Create a directory named `templates` and inside it, create an HTML file named `index.html`.
```bash
mkdir my_terminal_app
cd my_terminal_app
mkdir templates
cd templates
touch index.html
```
